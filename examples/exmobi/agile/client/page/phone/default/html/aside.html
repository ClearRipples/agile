侧边栏组件

左侧菜单栏:右滑可以唤起

push overlay reveal show close

右侧菜单栏:左滑可以唤起

push overlay reveal 自定义宽度
  • aside上的配置
  • 默认宽度为264px
  • data-position="left | right"
  • data-transition="push | overlay | reveal"
  • data-show-close="true | false"
  • push
  • 侧边栏与页面同时滑动
  • overlay
  • 侧边栏从页面上层划出
  • reveal
  • 侧边栏在底部，页面其上方划出
  • section上的配置
  • 默认在section上滑动操作不唤起左右侧菜单，通过设置data-aside-left和data-aside-right可以唤起
  • data-aside-left
  • data-aside-left="#aside id"右滑唤起左侧aside
  • data-aside-right
  • data-aside-right="#aside id"左滑唤起右侧aside
